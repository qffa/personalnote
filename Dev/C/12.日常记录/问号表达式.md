# C问号表达式

三目运算符

**格式**

```
/* 表达式1成立则执行表达式2，否则执行表达式3 */

(expression_1) ? (expression_2) : (expression_3);
```

**例子**

```
int a = 10;
int b = 20;

int c = (a > b) ? a : b;
printf("c: %d\n", c);
```

c的值为20

三目运算符是一个表达式

**c++**

在c++中，整个表达式返回的是变量本身，可以当左值

在c中，返回的是变量的值


```c
*((a > b) ? &a : &b) //与c++代码等效

```
