#变量的初始化与赋值


静态局部变量的初始化在编译阶段就完成了，调用函数时不起作用

```
int func(void)
{
  int a;    //调用函数时分配栈空间
  a = 100;  //执行数据搬移

  static int b = 200;   //编译阶段完成，分配空间并设初值，程序运行起来没这段代码
  b = 300;  //执行数据搬移
}
```


## 初始化

变量初始化是给变量开辟一段空间，并给值，只执行一次。

- 静态变量的初始化，由编译器完成，在可执行程序里面就有了
- 自动变量，在程序运行时初始化，进了栈空间

## 赋值

赋值是CPU需要执行的指令，将数据搬移至指定空间
