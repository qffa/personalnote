# vue脚手架工具

## 安装


**安装nodejs**

Linux用nvm装

Windows可以下载zip包，解压后，添加path环境变量

**安装cnpm**

```
npm i cnpm -g --registry https://registry.npm.taobao.org
```

**安装vue-cli**

```
cnpm i vue-cli -g
vue list
```


## 创建vue项目


**创建项目**

我在Windows下做的

```
$ vue init webpack myvue
```

填写完项目信息后，一路选No，下一步

**初始化并运行**

```
$ cnpm install
$ npm run dev
```
