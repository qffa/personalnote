# 进制转换

## 问题

二八十六进制转换及输出

## 解决方案

使用 `bin()`, `oct()`, `hex()` 等函数，输出字符串

```
>>> x = 1234
>>> bin(x)
'0b10011010010'
>>> oct(x)
'0o2322'
>>> hex(x)
'0x4d2'
>>>
```

使用 `format()` 函数

```
>>> format(x, 'b')
'10011010010'
>>> format(x, 'o')
'2322'
>>> format(x, 'x')
'4d2'
>>>
```

使用 `int()` 函数进行进制转换

```
>>> int('4d2', 16)
1234
>>> int('10011010010', 2)
1234
>>>
```

## 注意

八进制前缀是 `0o`

```
>>> os.chmod('script.py', 0o755)
>>>
```
