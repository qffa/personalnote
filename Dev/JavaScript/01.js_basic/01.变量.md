# 变量

## 定义变量

**方法**

- var：定义全局变量或局部变量（函数内部），尽量少用
- let：变量可被重新赋值
- const：变量不允许再被重新赋值

`let`和`const`在ES6(ES2015)中被添加，具有块级作用域

*let & const have a block-level scope*

**代码**

```js
let age = 30;
console.log(age);
age = 31;
console.log(age);
const score;    //error
const timeout = 30;
timeout = 10;   //error
```

**建议**：总是使用`const`，除非明确知道该变量需要被赋值

## 数据类型

**基本数据类型**

Primitive data types，数据被直接分配进内存。他们不是资源

- String
- Numbers
- Boolean
- null
- underfined
- Symbol(ES6 new)


**代码**

```js
const name = 'John';
const age = 30;
const rating = 4.5;
const isCool = true;
const x = null;
const y = undefined;
let z; //underfined

console.log(typeof name);
console.log(typeof z);
```

## 字符串

**字符串拼接**
```js
const name = 'John';
const age = 30;

//字符串拼接
console.log('My name is ' + name + ' and I am ' + age);
//模板，注意反引号
console.log(`My name is ${name} and I am ${age}`);

const hello = `My name is ${name} and I am ${age}`;

console.log(hello);
```

**常用函数**

```js
const s = 'Hello World';

console.log(s.length);
console.log(s.toUpperCase());
console.log(s.toLowerCase());

console.log(s.substring(0,5).toUpperCase());
console.log(s.split(''));


const s2 = 'technology, computers, it, code';
console.log(s2.split(', '));
```

## 对象

**object**

```js
const person = {
    firstName: 'John',
    lastName: 'Doe',
    age: 30,
    hobbies: ['music', 'movies', 'sports'],
    address: {
        street: '50 main st',
        city: 'Boston',
        state: 'MA'
    }
}

console.log(person.hobbies[2]);
console.log(person.address.city);

// 另一种f访问属性的方法方式
console.log(person["lastName"])

//添加属性
person.email = 'john@gmail.com';

console.log(person);

//解构赋值ES6
const {firstName, lastName, address:{ city }} = person;

console.log(firstName, city);
```

**解构赋值**

数组
```js
let [a,b] =[1,2]
```

对象
```js
var {foo,bar} = {foo: 'aaa', bar: 'bbb'};
```

**实现深拷贝**

```js
var obj = {
  name: 'FungLeo',
  sex: 'man',
  old: '18'
}

// ES6 扩展云算法实现对象的深拷贝
var { ...obj2 } = obj
```

数组也行

```js
var arr = [1,2,3,4,5]
var [ ...arr2 ] = arr
arr[2] = 5
console.log(arr)
console.log(arr2)
```

Object.assign

```js
const target = { a: 1, b: 2 };
const source = { b: 4, c: 5 };

const returnedTarget = Object.assign(target, source);

console.log(target);
// expected output: Object { a: 1, b: 4, c: 5 }

console.log(returnedTarget);
// expected output: Object { a: 1, b: 4, c: 5 }

// 等效？
for (let k in source) {
  target[k] = source[k]
}
```

**遍历对象**

使用`for in`

```js
obj = {
  name: 'jack',
  age: 18,
  gender: 'm'
}

for (let k in obj) {
  console.log(k);
  console.log(obj[k]);
}

//过滤继承属性
for (var key in obj) {
    if (obj.hasOwnProperty(key)) {
        console.log(key);
    }
}
```

也会把方法也遍历出来
