# 共享内存

共享内存就是OS在物理内存中开辟一大段缓存空间，不过与管道、消息队列调用`read`、`write`、`msgsnd`、`msgrcv`等API来读写所不同的是，使用共享内存通信时，进程是直接使用地址来共享读写的。

由于直接使用地址来读写缓存时，效率会更高，所以，如果进程涉及到超大量的数据通信时，必须使用“共享内存”这种直接使用地址操作的通信方式。

## 共享内存原理

多个进程中的虚拟地址空间（堆栈之间的share memory区）映射到同一块物理地址

1. 调用API，让OS在物理内存上开辟出一大段缓存空间
2. 让各自进程空间与开辟出的缓存空间建立映射关系


当多个进程映射并共享同一个空间时，在写数据的时候可能会出现相互干扰。此时往往需要加保护措施，让每个进程在没有操作时不要被别人干扰，等操作完以后，别的进程才能写数据

- 文件锁
- 信号量

## 共享内存使用步骤

1. 进程调用`shmget`函数创建新的或获取已有共享内存
2. 进程调用`shmat`函数，将物理内存映射到自己的进程空间
3. `shmdt`函数，取消映射
4. 调用`shmctl`函数释放开辟的那片物理内存空间


## API

### shmget

**函数原型**

```
#include <sys/ipc.h>
#include <sys/shm.h>

int shmget(key_t key, size_t size, int shmflg);
```

**功能**

创建新的，或者获取已有的共享内

- 如果`key`值没有对应任何共享内存，就创建一个新的共享内存
- 如果`key`值有对应某一个共享内存，就直接获取`key`所对应的共享内存

**返回值**

- 成功：返回共享内存的标识符，以后续操作
- 失败：返回`-1`，并且`errno`被设置。

**参数**

1. `key`：用于生成共享内存的标识符

    - 使用`IPC_PRIVATE`宏
    - 指定一个长整型数
    - 使用`ftok`函数

2. `size`：指定共享内存的大小，我们一般要求`size`是虚拟页大小的整数倍

    - 一般来说虚拟页大小是4k（4096字节）
    - 如果你指定的大小不是虚拟页的整数倍，也会自动帮你补成整数倍。

3. `semflg`：指定原始权限和`IPC_CREAT`

    - 比如`0664|IPC_CREAT`

**代码**

```
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>
#include <sys/ipc.h>
#include <sys/shm.h>
#include <sys/types.h>
#include <sys/types.h>
#include <sys/stat.h>
#include <fcntl.h>

#define SHM_FILE "./shmfile"
#define SHM_SIZE 4096


int shmid = -1;

void print_err(char *estr)
{
        perror(estr);
        exit(-1);
}

/* 封装共享内存创建函数 */
void create_or_get_shm(void)
{
        int fd = -1;
        key_t key = -1;

        /* 创建文件 */
        fd = open(SHM_FILE, O_RDWR|O_CREAT, 0664);
        if (fd == -1)
                print_err("open fail");

        /* 使用文件生成key值 */
        key = ftok(SHM_FILE, 'b');
        if (key == -1)
                print_err("ftok fail");

        /* 使用key创建共享内存 */
        shmid = shmget(key, SHM_SIZE, 0664|IPC_CREAT);
        if (shmid == -1)
                print_err("shmget fail");

}

int main(void)
{
        /* 调用共享内存创建函数 */
        create_or_get_shm();

        return 0;
}
```

### shmat

**函数原型**

```
#include <sys/types.h>
#include <sys/shm.h>

void *shmat(int shmid, const void *shmaddr, int shmflg);

int shmdt(const void *shmaddr);
```

**功能**

将`shmid`所指向的共享内存空间映射到进程空间（虚拟内存空间），返回映射后的起始地址（虚拟地址）

**返回值**

- 成功：则返回映射地址
- 失败：返回`（void *）-1`，并且`errno`被设置。

**参数**

1. `shmid`：共享内存标识符
2. `shmaddr`：指定映射的起始地址，两种指定方式
    - 自己指定映射的起始地址，一般不这么做
    - 指定为`NULL`，系统自动分配
3. `shmflg`：指定映射条件
    - `0`：以可读写方式映射
    - `1`：只写方式映射
    - `SHM_RDONLY`：只读

**代码**

进程1

```
int main(void)
{
        void *shmaddr = NULL;

        /* 穿件共享内存 */
        create_or_get_shm();

        /* 建立共享内存映射 */
        shmaddr = shmat(shmid, NULL, 0);
        if (shmaddr == (void *)-1)
                print_err("shmat fail");

        /* 封装测试数据 */
        char buf[300] = {"abcdefghijklmn......."};      

        while (1) {
                /* 写个整形数 */
                //(int *)shmaddr = 100;

                /* 向共享内存写一些数据 */
                memcpy(shmaddr, buf, sizeof(buf));
                sleep(1);

        }

        return 0;
}
```

进程2

```
int main(void)
{
        void *shmaddr = NULL;

        /* 创建或获取共享内存 */
        create_or_get_shm();

        /* 建立共享内存映射 */
        shmaddr = shmat(shmid, NULL, 0);
        if (shmaddr == (void *)-1)
                print_err("shmat fail");

        /* 创建用于存储数据的缓存buf */
        char buf[300] = {};     

        while (1) {
                /* 从共享内存读数据 */
                if(strlen((char *)shmaddr) != 0) {
                        printf("%s\n", (char *)shmaddr);
                        bzero(shmaddr, SHM_SIZE);   /* 共享内存空间清零 */
                }
        }

        return 0;
}
```

### shmdt
删除映射

**函数原型**

```
#include <sys/types.h>
#include <sys/shm.h>

int shmdt(const void *shmaddr);
```

**功能**

取消建立的映射

**返回值**

调用成功返回`0`，失败返回`-1`，且`errno`被设置

**参数**

`shmaddr`：映射的起始地址（虚拟地址）


### shmctl

**函数原型**

```
#include <sys/ipc.h>
#include <sys/shm.h>

int shmctl(int shmid, int cmd, struct shmid_ds *buf);
```

**功能**

根据`cmd`的要求，对共享内存进行相应控制

- 获取共享内存的属性信息
- 修改共享内存的属性信息
- 删除共享内存
- 等等

**参数**

1. `shmid`：标识符
2. `cmd`：控制选项
    - `IPC_STAT`：从内核获取共享内存属性信息到第三个参数（应用缓存）
    - `IPC_SET`：修改共享内存的属性，修改方法与消息队列相同
    - `IPC_RMID`：删除共享内存，只有当所有的映射取消后，才能删除共享内存
3. `buf`：类型为`struct shmid_ds`
    - `cmd`为`IPC_STAT`时，`buf`用于存储原有的共享内存属性
    - `cmd`为`IPC_SET`时，`buf`中放的是新的属性设置
4. 等等

**返回值**

调用成功`0`，失败则返回`-1`，并且`errno`被设置

**struct shmid_ds结构体**

可以从man手册查看到

```
struct shmid_ds {
    struct ipc_perm shm_perm;    /* Ownership and permissions */
    size_t          shm_segsz;   /* Size of segment (bytes) */
    time_t          shm_atime;   /* Last attach time */
    time_t          shm_dtime;   /* Last detach time */
    time_t          shm_ctime;   /* Last change time */
    pid_t           shm_cpid;    /* PID of creator */
    pid_t           shm_lpid;    /* PID of last shmat(2)/shmdt(2) */
    shmatt_t        shm_nattch;  /* No. of current attaches */
    ...
};

struct ipc_perm {
    key_t          __key;    /* Key supplied to shmget(2) */
    uid_t          uid;      /* Effective UID of owner */
    gid_t          gid;      /* Effective GID of owner */
    uid_t          cuid;     /* Effective UID of creator */
    gid_t          cgid;     /* Effective GID of creator */
    unsigned short mode;     /* Permissions + SHM_DEST and
                                SHM_LOCKED flags */
    unsigned short __seq;    /* Sequence number */
};

```

## 改进共享内存数据读取

**以上例子中的缺陷**

1. `strlen`函数只能用于判断字符串，如果是结构体、整形、浮点型数据，`strlen`将无法判断
2. 没有数据时，cpu会一直循环的判断，浪费cpu资源

**改进方法**

当共享内存没有数据时，读进程休眠，当写进程把数据写完后，将读进程唤醒，也就是同步。

实现同步的方法：
- 方法1：使用信号signal来实现
- 方法2：使用信号量semaphore实现

**代码**

使用信号实现

进程1

```

#include <stdlib.h>
#include <unistd.h>

#include <string.h>

#include <errno.h>


#define SHM_FILE "./shmfile"
#define SHM_SIZE 4096


int shmid = -1;
void *shmaddr = NULL;

void print_err(char *estr)
{
        perror(estr);
        exit(-1);
}


void create_or_get_shm(void)
{
        int fd = -1;
        key_t key = -1;

        fd = open(SHM_FILE, O_RDWR|O_CREAT, 0664);
        if (fd == -1)
                print_err("open fail");

        key = ftok(SHM_FILE, 'b');
        if (key == -1)
                print_err("ftok fail");

        shmid = shmget(key, SHM_SIZE, 0664|IPC_CREAT);
        if (shmid == -1)
                print_err("shmget fail");

}


void signal_fun(int signo)
{
        shmdt(shmaddr);
        shmctl(shmid, IPC_RMID, NULL);
        remove("./fifo");
        remove(SHM_FILE);

        exit(-1);
}


int get_peer_pid(void)
{
        int ret = -1;
        int fifofd = -1;

        /* 创建无名管道 */
        ret = mkfifo("./fifo", 0664);
        if (ret == -1 && errno != EEXIST)
                print_err("mkfifo fail");

        /* 打开管道文件 */
        fifofd = open("./fifo", O_RDONLY);
        if (fifofd == -1)
                print_err("open fifo fail");

        int peer_pid;
        /* 读无名管道中的数据，另一个进程发过来的PID */
        ret = read(fifofd, &peer_pid, sizeof(peer_pid));
        if (ret == -1)
                print_err("read fifo fail");

        //printf("peer pid: %d\n", peer_pid);
        return peer_pid;
}


int main(void)
{
        /* 注册信号处理函数 */
        signal(SIGINT, signal_fun);

        /* 创建共享内存 */
        create_or_get_shm();

        /* 映射共享内存 */
        shmaddr = shmat(shmid, NULL, 0);
        if (shmaddr == (void *)-1)
                print_err("shmat fail");

        /* 封装测试数据 */
        char buf[300] = {"abcdefghijklmn......."};      

        int peer_pid = -1;
        /* 获取读进程的PID */
        peer_pid = get_peer_pid();

        while (1) {
                /* 向共享内存写一些数据 */
                memcpy(shmaddr, buf, sizeof(buf));

                /* 向读进程发送信号，唤醒读进程 */
                kill(peer_pid, SIGUSR1);
                sleep(1);
        }

        return 0;
}
```

```
                /* 空捕获，唤醒pause函数 */

void send_self_pid(void)
{
        int ret = -1;
        int fifofd = -1;

        mkfifo("./fifo", 0664);
        if (ret == -1 && errno != EEXIST)
                print_err("mkfifo fail");

        fifofd = open("./fifo", O_WRONLY);
        if (fifofd == -1)
                print_err("open fifo fail");

        int pid = getpid();
        /* 向管道中写入自己的PID */
        ret = write(fifofd, &pid, sizeof(pid));
        if (ret == -1)
                print_err("write fifo fail");

}

        /* 创建共享内存 */
        create_or_get_shm();

        /* 建立共享内存映射 */
```
