# vim 配置文件

file location: ~/.vimrc

```
"this is comment

set number
set autoindent
set tabstop=4
set smarttab
set softtabstop=4
set shiftwidth=4
set expandtab
set smartindent
set hlsearch

set cursorline

filetype on
filetype indent on
filetype plugin on
syntax on

autocmd FileType yaml,html setlocal ai ts=2 sw=2 et

inoremap jk     <ESC>
inoremap (      ()<Esc>i
inoremap [      []<Esc>i
inoremap {      {}<Esc>i
inoremap '      ''<Esc>i
inoremap "      ""<Esc>i
inoremap <c-o>  <Esc>o
inoremap <c-a>  <Esc>A
inoremap <c-l>  <Right>
inoremap <c-h>  <Left>

execute pathogen#infect()
map <F10> :NERDTreeToggle<CR>
```
