# vim可视化模式

**选择**

| Key      | Description   |
|:---------|:--------------|
| `v`      | select char   |
| `V`      | select line   |
| `ctrl+v` | select block  |
| `gv`     | repeat select |


**相关操作**

| Key  | Description |
|:-----|:------------|
| `gu` | 转小写      |
| `gU` | 转大写      |
| `g~` | 大小写互转  |
| `n>` | 右移n次     |
| `n<` | 左移n次     |
