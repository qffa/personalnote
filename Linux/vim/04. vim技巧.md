# vim技巧

| Key      | Description                 |
|:-------- |:--------------------------- |
| `.`      | repeat the last operation   |
| `*`      | search word at cursor       |
| `#`      | same with upper             |
| `ctrl+o` | return to last persion      |
| `zz`     | make current line in middle |



说明

```
* 查找光标所在处的单词，向下查找
# 查找光标所在处的单词，向上查找
```
